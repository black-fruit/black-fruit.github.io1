Hexo版本

下载和启用主题

转到Releases页面选择最新版本下载

打开Hexo根目录下的_config.yml文件

把Theme项改成下载的主题文件夹名字

配置主题

配置主题文件

打开主题根目录下的_config.yml文件

先转到侧栏和顶栏位置修改，然后修改gitalk项

安装npm插件

在主题根目录下启动GitBash输入：

npm install hexo-generator-search --save
